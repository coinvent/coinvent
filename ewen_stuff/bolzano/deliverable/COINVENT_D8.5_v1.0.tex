This deliverable presents the implemented APIs for the Coinvent system. It presents some examples and explains how access the APIs on an existing server, and how to install them locally so that the APIs can be accessed without the need for the internet. RESTful http web services have been implemented for HDTP, HETS and the amalgams system. The system allows iterative blending to take place by re-inserting computed blends as input theories.
