Instructions

This page explains using the CoInvent system to compute blends. This image annotates the fields and their function. In order to use any of the possible funcitonality, input space must be defined in the Concept Editors, which can use
  • Previously defined: previously defined or referenced concepts are saved in the drop down list. These can be viewed and edited using the "Show concept" button.
  • From url:enter the url of a concept file
  • Enter text:directly write and name a concept in CASL or OWL format
The combo box has four options - these are listed below:

  • Using HDTP

    In order to use HDTP and HETS to compute blends, two input concepts are required. You can use the concept editors to choose a concept file as described above. The input language must be CASL. The result is demonstrated in the text area named "Computed Blend". The HETS development graph can be viewed and inspected, and the blend save as a new concept using the save button. The next computed result is given by pressing the "next" button. The "close" button will end the process manually. See the examples page for a list of concepts online which can be used.

  • Use Amalgams with CASL

    In order to use this version of Amalgams to compute blends, a single input theory in CASL format is specified, and the names of the input space names specified. Press the "Compute Blend button". The best result, as computed by amalgams, is given first and the subsequent results can be found using the "next" button. See the examples page for a list of concepts online which can be used.

  • Use Amalgams with OWL

    In order to use this version of Amalgams to compute blends, a single input theory in CASL format is specified, and the names of the input space names specified. Press the "Compute Blend button. The best result, as computed by amalgams, is given first and the subsequent results can be found using the "next" button. See the examples page for a list of concepts online which can be used.

  • Use the haiku generation module

    In order to use the conceptual blending machinery to generate haikus, enter two concepts of any form into the concept editors. Then press the "Compute Blend" button to generate a haiku.
